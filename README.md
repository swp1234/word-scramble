# Word Scramble - 단어 퍼즐 게임

뒤섞인 알파벳을 올바른 순서로 배치하는 재미있는 단어 퍼즐 게임입니다.

## 게임 특징

### 게임플레이
- **난이도 선택**: Easy (4글자) → Medium (5-6글자) → Hard (7+글자) → Mixed
- **타이머 시스템**: 단어당 30초 제한 시간
- **라이프 시스템**: 3개 라이프로 시작, 오답 시 라이프 감소
- **점수 시스템**: 남은 시간 × 단어길이 × 콤보 배율
- **콤보 시스템**: 연속 정답으로 보너스 배율 증가
- **레벨 시스템**: 10개 문제씩 한 레벨, 레벨업 시 새 도전

### UI/UX
- **다크 모드**: 기본 테마 (#0f0f23)
- **마젠타 테마**: 주요 색상 (#e056fd)
- **Glassmorphism**: 모던한 반투명 효과
- **반응형 디자인**: 모든 화면 크기 지원 (360px~)
- **마이크로인터랙션**: 부드러운 애니메이션과 피드백

## 게임 플로우

1. **인트로 화면**: 난이도 선택
2. **게임 화면**:
   - 뒤섞인 글자 표시
   - 글자 선택하여 답 구성
   - 30초 타이머
   - 힌트 사용 가능
3. **결과 화면**: 레벨 점수, 통계, AI 분석
4. **게임 오버**: 최종 점수, 재도전 옵션

## 점수 계산

```javascript
점수 = (남은 시간) × (단어 길이) × (콤보 배율)
콤보 배율 = 1 + (연속 정답 수 × 0.1)
```

### 예시
- 20초 남음, 5글자 단어, 콤보 3배율:
  - 20 × 5 × (1 + 3 × 0.1) = 20 × 5 × 1.3 = 130 포인트
